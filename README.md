# MERN Blog Web Application

## Overview

The MERN Blog Web Application is a full-stack blogging platform developed using the MERN stack — MongoDB, Express.js, React.js, and Node.js. The project is designed to provide users with a modern and responsive blogging experience where they can create, edit, manage, and delete blog posts efficiently.

This project demonstrates the practical implementation of full-stack web development concepts including REST API integration, frontend-backend communication, authentication handling, database management, and responsive UI development.

---

# Features

## User Features

* User Registration and Login Authentication
* Secure Authentication System
* Create New Blog Posts
* Edit Existing Blog Posts
* Delete Blog Posts
* View All Blogs
* Responsive User Interface
* Dynamic Routing Using React Router
* REST API Integration
* Real-Time Frontend and Backend Communication
